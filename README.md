# GESP Hardware Assets

This repository stores assets (images, diagrams) for the GESP hardware course.

## Directory Structure

- `images/hardware/scenarios/` - Hero images for course covers (16:9, WebP/JPEG)
- `images/hardware/diagrams/` - Wiring diagrams (PNG/SVG)
